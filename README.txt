============================================
   دليل تشغيل معين - المساعد الصوتي
============================================

التعديلات على المشروع:
-----------------------
✅ تم تفعيل زر "معين" في dashboard.html
✅ تمت إضافة نافذة منبثقة للتسجيل الصوتي
✅ تم إضافة خادم Python لتحويل الصوت لنص


طريقة التشغيل:
---------------

الخطوة 1: تثبيت Python
- حمّل Python 3.8 أو أحدث من python.org

الخطوة 2: تشغيل الخادم
- Windows: اضغط دبل كليك على start_server.bat
- أو نفذ: python voice_server.py

الخطوة 3: استخدام معين
- افتح ملف dashboard.html في المتصفح
- اضغط على زر "معين" في الأعلى
- اضغط "ابدأ التسجيل"
- تحدث بالعربية أو الإنجليزية
- اضغط "إيقاف التسجيل"
- شاهد النص يظهر!


ملاحظات:
--------
- أول تشغيل يحمّل نموذج Whisper (2-5 دقائق)
- بعد ذلك يعمل حتى بدون إنترنت
- يدعم العربية والإنجليزية تلقائياً


الملفات المضافة:
----------------
✅ voice_server.py - الخادم
✅ requirements.txt - المكتبات المطلوبة
✅ start_server.bat - تشغيل سريع (Windows)
✅ README.txt - هذا الملف


المشاكل الشائعة:
----------------
1. "لا يمكن الوصول للميكروفون"
   → اسمح للمتصفح بالوصول للميكروفون

2. "خطأ في الاتصال بالخادم"
   → تأكد من تشغيل voice_server.py

3. "No module named whisper"
   → نفذ: pip install -r requirements.txt


للدعم: راجع التعليمات داخل ملف voice_server.py
